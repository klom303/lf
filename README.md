# lf - A sample MVC PHP framework

> 开发这个框架的目的是为了查漏补缺、夯实基础，见证自己的成长经历。

[我的博客实例](http://www.yejialu.com/)

目前框架实现了

1. 实现了简单的MVC模式  
2. 实现了符合PSR-4的自动载入（目前还没有加入composer）  
3. 实现了简单的自定义路由控制  
4. 实现了基于mysql pdo的简单的查询构造器，可链式调用  
5. 实现了一个简单的command组件，可以通过命令行执行设置的任务  
6. 实现了通用的分页组件  

还在缓慢开发中...
